## üé® Design Guidelines

### Color Palette
- **Primary**: Blue (#007AFF) - Apple system blue
- **Secondary**: Orange (#FF9500) - Premium features
- **Success**: Green (#34C759) - Enabled states
- **Background**: White (#FFFFFF) or System background
- **Text**: Black (#000000) or System labels

### Typography
- **Headings**: SF Pro Display, Bold, 28-34pt
- **Body**: SF Pro Text, Regular, 16-18pt
- **Captions**: SF Pro Text, Regular, 14pt
- **Emphasis**: SF Pro Text, Semibold

### Status Bar
- Show full signal (5 bars)
- WiFi connected
- Battery at 80-100%
- Time: 9:41 AM (Apple convention)

### Navigation
- Back button with app name
- Title centered
- Action buttons (‚öôÔ∏è, ‚ûï) in nav bar

### Accessibility
- High contrast text
- Large tap targets (44x44pt minimum)
- Clear icons with labels
- No reliance on color alone

1. Create artboard at exact size (1290√ó2796)
2. Import app UI screenshots
3. Add overlays, captions, backgrounds

1. Take screenshots on physical iPhone 15 Pro Max

## ‚úÖ Screenshot Quality Checklist

For each screenshot, verify:
- [ ] Correct resolution (1290√ó2796 for 6.7")
- [ ] No blurriness or artifacts
- [ ] Status bar looks realistic
- [ ] Text is readable at thumbnail size

- [ ] Demonstrates actual functionality
- [ ] Follows App Store guidelines (no comparison to competitors)

## üö´ What to Avoid

‚ùå **Don't Include**:
- Placeholder or dummy content
- "Coming soon" features
- Competitor comparisons
- Adult content or sensitive imagery
- Prices (except in IAP screenshots)
- Device frames (Apple adds automatically)
- Excessive text overlays
- Low-resolution images
- Inconsistent branding

‚úÖ **Do Include**:
- Actual app UI
- Real functionality
- Clear feature demonstrations
- Consistent design language
- High-quality assets
- Readable text
- Compelling visuals
